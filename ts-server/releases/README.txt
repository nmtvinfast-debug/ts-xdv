Đặt bản cài TS-XDV tại đây (phục vụ qua https://<máy-chủ>/releases/...)

  web/              → copy toàn bộ thư mục build/web (Flutter)
  ts_xdv.exe        → file cài Windows (build/windows/x64/runner/Release/)
  ts-xdv.apk        → file Android (build/app/outputs/flutter-apk/)

iOS: dán link TestFlight/App Store trong Admin → Cấu hình → Cập nhật app.

Mặc định trong cấu hình:
  Web:     /releases/web/
  Windows: /releases/ts_xdv.exe
  Android: /releases/ts-xdv.apk
